YaST2 Module SAPMedia

This module is used for preparing the installations envinroment for a SAP product.
The installation envinroment will be build in in /data/SAP_INST/<Product Count> directory.
Except of the installation master all SAP media will be copied into /data/SAP_CDs.
The installation master must be saved into /data/SAP_INST/<Product Count>/Instmaster 
because the sap installer makes changes in this directory for all installation.

The SAPMedia Module can also provide the medias in /data/SAP_CDs as installation
server via NFS for other SAP product installations. For this reason the SAP installation
master will be copied into /data/SAP_CDs/Instmaster.

The SAPMedia Module provides an autoyast client to build up the installations envinroments for a SAP products.
The section of the sap-media part of auoyast has following syntax:

<sap-media config:type="list">
  <product> 
    <count config:type="integer">0</count>
    <tocopy config:type="boolean">true</tocopy>
    <medias config:type="list">
      <media>
        <url>nfs://server/path1</url>
        <type>sap</type>
      </media>
      <media>
        <url>nfs://server/path2</url>
        <type>sap</type>
      </media>
      <media>
        <url>nfs://server/path3</url>
        <type>supplement</type>
      </media>
    </medias>
  </product> 
  <product> 
    <count config:type="integer">1</count>
    <tocopy config:type="boolean">true</tocopy>
    <medias config:type="list">
      <media>
        <url>nfs://server/path1</url>
        <type>sap</type>
      </media>
      <media>
        <url>nfs://server/path2</url>
        <type>sap</type>
      </media>
      <media>
        <url>nfs://server/path3</url>
        <type>supplement</type>
      </media>
    </medias>
  </product> 
</sap-media>
